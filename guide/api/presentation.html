# 🔌 Presentation

This is the documentation about the API of ComexposiumConnect.

The API scheme is HTTPS and is based on JWT and Kuzzle.

The content-type expected is application/json.

# Pre-production

Base url for the API is https://api.preprod.comexposium-sso.com

# Production

Base url for the API is https://api.comexposium-sso.com

# Sandbox

You can test the API on our sandbox which is plugged on the pre-production environment
