# 🔭 Get user items

Get list of items in favorites.

# Description

ENDPOINT DESCRIPTION METHOD EXEMPLE
/_plugin/Comexposium/mySelection/get?sessionSalon={sessionSalon} get list of items in favorites GET https://api.preprod.comexposium-sso.com/_login

# Call Exemple

curl -X GET "https://api.preprod.comexposium-sso.com/_plugin/Comexposium/mySelection/get?sessionSalon=mysession" -H "accept: application/json" -H "Authorization: Bearer mytoken"

# Details

# Responses

You can only get a 200 or 401 status code by using this endpoint, you will find results in the statusCode attribute.
